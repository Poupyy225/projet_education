À Propos de cette Plateforme

🎯 Objectif du Projet

Cette plateforme interactive a été développée dans le cadre d'un projet de semestre visant à démocratiser l'accès aux tests statistiques non paramétriques.

📊 Tests Implémentés
  • Test de Wilcoxon pour données appariées
  • Test de Mann-Whitney pour échantillons indépendants
  • Test de Kruskal-Wallis pour comparaisons multiples
  • Corrélation de Spearman
  • Test de Friedman pour blocs randomisés
🛠️ Technologies Utilisées
Backend:
  • Flask (Python)
  • Pandas pour la manipulation des données
  • SciPy pour les tests statistiques
  • Matplotlib/Seaborn pour les visualisations
Frontend:
  • HTML5/CSS3/JavaScript
  • Bootstrap 5 pour le design
👨‍🎓 Équipe de Développement

Projet réalisé dans le cadre du cours de statistiques non paramétriques.

Note pédagogique : Cette plateforme est conçue à des fins éducatives et ne remplace pas l'expertise d'un statisticien professionnel.
